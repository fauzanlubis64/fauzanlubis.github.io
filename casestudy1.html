Redesain Aplikasi Peduli Lindungi

Latar Belakang Permasalahan

Tujuan dibuat nya aplikasi Peduli lindungi adalah untuk mempermudah masyarakat untuk mengetahui angka penyebaran covid-19 yang ada di sekitar kita. tapi juga dapat dimanfaatkan agar masyarakat tetap beraktifitas di tengah pandemi dengan mengedepankan protokol kesehatan.

Peran dari Aplikasi Peduli Lindungi tidak hanya untuk melacak penyebaran covid, namun juga bisa mendapatkan informasi vaksin, download sertifikat vaksin, dan dokumen perjalanan seperti masuk ke mall, supermarket dengan hanya menscan sertifikat vaksin kita jika kita sudah melakukan vaksinasi. dan aplikasi ini juga bisa mengetahui informasi hasil tes covid-19 apakah itu negatif atau positif.

Nah, di aplikasi peduli lindungi sekarang terdapat masalah yang sering dihadapi user/masyarakat dalam menggunakan aplikasi tersebut, seperti:

  1. Fitur Scan , dimana fitur scan ini terdapat di bagian atas layar, sehingga user kesulitan ketika menekan tombol scan nya, dikarenakan kebanyakan dari kita sering menggunakan hp hanya dengan 1 tangan.

  2. Banyak fitur yang jarang sekali dipakai oleh user yang ditampilkan dilayar utama/home, seperti teledokter, aturan perjalanan, statistik, cari kamar, EHAC, pelayanan kesehatan dan fitur cari kamar.
  3. Fitur hasil tes covid-19, di mana seharusnya di tampilkan langsung di layar home, supaya mempermudah user untuk melihat langsung tanpa melakukan banyak tab sehingga memakan waktu.
  4. Sertifikat vaksin sebaiknya ditampilkan di layar home, gunanya apa? yang pasti akan mempermudah user untuk melihat informasi sertifikat vaksin tanpa harus melakukan banyak tab seperti sekarang ini. untung-untung jaringan nya bagus jika tidak pasti akan mempersulit user untuk mendapatkan informasi vaksin.
  5. Fitur check in ketika masuk ke gedung, dimana seharusnya ditampilkan di layar home saja karena mempermudah user untuk mengetahui riwayat chekin nya ketika masuk ke gedung, tanpa melakukan banyak tab/step karena terlalu ribet ketika kita melakukan banyak tab/step.

Fitur Utama Aplikasi Peduli Lindungi

  1. Scan Qr-Code
  2. Informasi Sertifikat Vaksin
  3. Melihat Informasi Hasil Test Covid

Keterangan

  1. fitur-fitur yang jarang sekali terpakai akan disembunyikan di dalam icon more, sehingga tampilan home diperuntukkan untuk fitur yang sering digunakan
  2. Sertifakat vaksin langsung muncul dihalaman home, sehingga mempermudah user untuk menunjukkan nya kepada petugas ketika memasuki gedung tanpa banyak melakukan banyak step, ketika user ingin mendownload sertifikat nya user hanya mengklik sekali saja langsung diarahkan ke halaman download sertifikat vaksin
  3. Hasil tes covid-19 akan muncul jika kita sudah melakukan tes maupun itu swab test ataupun PCR. informasi positif atau negatif nya akan terlihat langsung, yang pasti akan memudahkan user untuk mendapatkan suatu informasi test covid-19
  4. Riwayat check-in setelah kita masuk ke gedung akan muncul di halaman home, sehingga memudahkan kita untuk melihat langsung tenmpat mana saja yang sudah kita masuki, tanpa harus melakukan banyak tab/step.
